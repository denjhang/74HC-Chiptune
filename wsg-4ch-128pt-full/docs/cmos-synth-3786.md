# 古董电子琴架构笔记 — 模数结合的两种经典路线

> 记录两类 80 年代国产/进口电子琴的设计思路:
> - **JDQ49A**(国产佳莺):全分立 NPN 三极管,数字分频 + 模拟包络
> - **3786 型**:CD4000 系列 CMOS 极简,加法合成 + 电容包络
>
> 这两种路线跟本项目(74HC-Chiptune)的"纯数字 TDM + 微码"是**三个坐标系**,
> 但思路可以互相借鉴——尤其是用**模拟元件(电容、电阻、三极管)替代数字状态机**,
> 能显著减少数字芯片数量。

## 1. 背景:三条合成器考古路线

| 路线 | 代表机型 | 哲学 | 有源元件 |
|------|---------|------|---------|
| **极繁(分立)** | 国产佳莺 JDQ49A(80 年代) | 全分立 NPN 三极管(3DG201 几十个) | 0 片 IC,几十个三极管 |
| **极简(CMOS)** | 3786 型电子琴 | CD4000 系列 CMOS 3-4 片 + 电容包络 | 3-4 片 CMOS |
| **现代数字** | 本项目(74HC-Chiptune) | 74HC + 微码 ROM + RAM | 25 片 74HC |

**核心观察**:JDQ49A 和 3786 都是**模数结合**——数字部分只做"分频/键选"这种纯组合逻辑,
其他**包络、滤波、混音全部交给模拟元件**。本项目现在反而把所有功能都塞进数字侧,
其实可以反过来借鉴古董琴的思路。

---

## 2. 路线 A:JDQ49A — 分立三极管的"模数结合"

### 2.1 整机架构

JDQ49A 是国产老电子琴里的传奇——**0 片 IC,几十个 3DG201(NPN 通用三极管)堆出来**。
8 复音 + 鼓机,完全是"手搓合成器"。

```
┌────────────────────────────────────────────────────────┐
│  数字部分(分立三极管搭)                                │
│    主振荡器(RC + 2 三极管)                            │
│        ↓                                               │
│    分频链:每个 flip-flop = 2 三极管 + RC               │
│      ├─÷2→ 高八度                                      │
│      ├─÷2→ 中八度                                      │
│      └─÷2→ 低八度                                      │
│        ↓                                               │
│    键盘矩阵选哪个八度哪个音                            │
└────────────────────────────────────────────────────────┘
                       ↓ 方波群
┌────────────────────────────────────────────────────────┐
│  模拟部分(分立三极管 + RC)                            │
│    音色:RC 低通滤波(方波→"圆润"波形)                │
│    包络:三极管 + 电容 = AR/ADSR                       │
│    VCA:三极管可变增益级                               │
│    输出:变压器耦合 / 直接驱动喇叭                     │
│    鼓机:噪声(三极管 BE 结反向击穿) + 单稳态 + 带通   │
└────────────────────────────────────────────────────────┘
```

### 2.2 数字部分的"三极管逻辑"

每个 flip-flop(分频器、触发器)都是**分立三极管搭的**:
- 2 个 3DG201 + 几个电阻电容 = 1 个双稳态
- 几十个双稳态级联 = 整个分频链
- 工作原理跟用 74HC74 完全一样,只是用三极管搭出来

**这就是为什么叫"模数结合"**——同一批三极管,有的工作在开关态(数字分频),
有的工作在线性态(模拟放大/滤波)。

### 2.3 模拟部分的精华

JDQ49A 的音色/包络完全靠模拟:

- **音色链**:方波 → 多级 RC 低通,转折频率决定"刺耳/圆润"
- **包络**:三极管 + 电容,充放电时间常数 = attack/release(详见第 4 节)
- **鼓机**:
  - 噪声源 = 三极管 BE 结反向偏置击穿(白噪声)
  - 底鼓 = 单稳态触发低频谐振回路
  - 军鼓 = 噪声 + 包络调制
  - 全分立,没有 555/CMOS 这种"作弊"

### 2.4 对本项目的启发

JDQ49A 证明:**纯分立元件就能做出复音 + 鼓机的合成器**。
我们的 74HC 项目其实跟 JDQ49A 是**同一种哲学的现代版**——
JDQ49A 用三极管搭触发器,我们用 74HC 集成触发器;
JDQ49A 用 RC 做滤波,我们用数字 wavetable ROM。

差异:**JDQ49A 的包络是模拟电容,我们的包络是数字 RAM 表**。如果未来要简化,
可以考虑回到电容包络(见第 4 节)。

## 3. 路线 B:3786 — CD4000 CMOS 极简

### 3.1 整机架构

3786 是另一个极端——**几片 CMOS 就搞定**,完全没有分立三极管的"堆料"。

```
┌────────────────────────────────────────────────────────┐
│  CD4000 系列 CMOS (3-4 片)                             │
│                                                        │
│   CD4046 (PLL) ──主振荡──→ CD4040 (纹波计数器)        │
│                              ├─÷2→ 高八度              │
│                              ├─÷2→ 中八度              │
│                              └─÷2→ 低八度              │
│                                  ↓                     │
│   CD4051 (8 选 1 模拟开关) ←── 键盘矩阵                │
│                                  ↓                     │
│   CD4069UB(线性区当运放)── RC 滤波/缓冲               │
│                                  ↓                     │
│   加法合成(电阻权重叠加)                              │
│                                  ↓                     │
│   CD4011/4093(鼓机单稳态 + 噪声)                      │
└────────────────────────────────────────────────────────┘
```

### 3.2 各 IC 的精妙用法

| IC | 在 3786 里的角色 |
|----|------------------|
| **CD4040 / CD4024**(纹波计数器) | 一片搞定 JDQ49A 几十个三极管搭的分频链 |
| **CD4051**(8 选 1 模拟开关) | 复音键选,选哪个八度的哪个音 |
| **CD4046**(PLL 锁相环) | 主振荡器,VCO 跟分频链锁相,稳定又能扫频 |
| **CD4011 / CD4093**(四 2 输入与非,4093 是施密特触发版) | 鼓机单稳态、噪声触发、包络整形 |
| **CD4069UB**(六反相器,**无缓冲版**) | 偏置线性区当模拟运放——RC 振荡器、带通滤波、VCA 全用它 |

**经典骚操作**:CD4069UB 当线性运放。无缓冲版反相器偏置在线性区(输入接 VDD/2),
就能当反相放大器用——这是 CMOS 数字 IC 当模拟用的标志技巧。

### 3.3 加法合成的极简实现

```
分频输出 1 ──[R1]──┐
分频输出 2 ──[R2]──┼──→ 求和节点 ──→ CD4069UB 缓冲 ──→ 输出
分频输出 3 ──[R3]──┘
```

不同权值电阻把多个方波(不同八度)叠加 = 加法合成。
本质就是 **R-2R DAC 的雏形**,只是"数字"输入是各方波而不是数字总线。

**对本项目的启发**:我们 74HC 项目用 4 通道 TDM + 单 DAC + RC 低通做"数字加法",
跟 3786 的"模拟电阻加法"是**同一思想的两种实现**——TDM 省了求和电阻但需要时钟同步,
模拟加法省了同步但需要精确电阻。

### 3.4 鼓机的极简实现

- **底鼓**:CD4046 VCO 触发后频率从高滑到低(sweep)→ "咚"
- **军鼓/踩镲**:CD4093 振荡器 + 噪声源 + 包络调制振幅
- 全部 2-3 片 CMOS 搞定

## 4. 核心借鉴:电容包络(SGS M208B1)

> 这是对本项目**最有启发**的一点。SGS-Thomson **M208B1**(80 年代电子琴专用 IC)
> 的包络**完全靠电容实现**,片内/片外没有 ADSR 状态机,没有 RAM 存包络参数。
> JDQ49A 的三极管包络、3786 的 RC 包络也是同一思路。

### 4.1 电容包络原理

```
键按下:
    ┌─ V_charge ──[R_attack]──┐
    │                           │
    │                          ─┴─ C_env
    │                          ─┬─
    │                           │
    └───────────────────────────┘
                              ↓
                        V_env(t) = V_charge × (1 - e^(-t/RA·C))

键松开:
    C_env ──[R_release]── GND
        V_env(t) = V_peak × e^(-t/RR·C)
```

- **Attack 时间** = RA × C(充电时间常数)
- **Release 时间** = RR × C(放电时间常数)
- V_env 直接驱动 VCA(模拟乘法器/OTA)

### 4.2 电容包络的优势

1. **0 状态机**——没有 RAM 存包络表,没有 FSM 跑包络曲线
2. **0 时钟依赖**——不依赖主时钟,时间常数纯由 RC 决定
3. **真实指数曲线**——电容充放电天然指数,比数字 ADSR 的折线段更"乐器感"
4. **响应即时**——按下键瞬间开始充电,不需要查表

### 4.3 电容包络的劣势

1. **不可编程**——RA/RR 是固定电阻,换包络要换元件
2. **通道独立**——每个发声通道需要独立的 C_env + 充放电开关,通道多时元件数爆炸
3. **温度漂移**——电容/电阻值随温度变化,包络时间会漂

### 4.4 对 74HC-Chiptune 项目的启发

我们现在 v1.4 的钢琴包络是**纯数字实现**:
- RAM[ch*8+4] 存 vol(4-bit)
- testbench 周期性 spfm_write 改 vol
- vol 直接进 wavetable ROM 地址线(查预乘音量表)

**如果改成电容包络**:
- 数字侧只保留 phase_acc / phase_step
- 音量从 RAM 移出,改用一个**模拟电压 V_env**(电容上的电压)
- DAC 输出 × V_env(用模拟乘法器,或 OTA)
- 键按下 → 触发充电开关(一个 CD4066 / 三极管)
- 键松开 → 触发放电开关

**省下的数字芯片**:
- 不再需要 vol 寄存器(reg_c 377 #5)
- 不再需要包络更新逻辑(testbench 里那一大堆 set_vol)
- 不再需要 wavetable ROM 里 16 级音量预乘 → ROM 容量 ÷16(8KB → 512B)

**代价**:
- 每通道一个电容 + 充放电开关 + 模拟乘法器(或共享)
- 增加模拟元件

**对 4 通道项目**:可能不划算(数字 RAM 才用 32B,模拟侧要 4 套电容)。
**对 8+ 通道项目**:可能划算(数字 RAM 还小,但包络状态机和 vol 寄存器会膨胀,模拟侧元件数线性增加)。

## 5. 思路汇总:本项目可以借鉴什么

| 借鉴点 | 来源 | 实现方式 | 省下什么 |
|--------|------|---------|---------|
| **电容包络** | SGS M208B1 / JDQ49A / 3786 | C_env + 充放电开关 + 模拟乘法 | vol 寄存器、包络状态机、ROM 音量预乘 |
| **PLL 主振荡** | CD4046 (3786) | 主频锁定到精确分频链 | 不需要稳定晶体振荡器(可用更便宜的 RC 主振) |
| **CD4069UB 当运放** | 3786 / 经典骚操作 | CMOS 反相器偏置在线性区 | 省专用运放 IC |
| **加法合成** | 3786 电阻权重 | 多路方波经加权电阻叠加 | 省求和 IC(我们用 TDM 数字求和,异曲同工) |
| **三极管分频** | JDQ49A | flip-flop = 2 三极管 + RC | 0 片 IC(但元件数爆炸,本项目用 74HC161 替代) |
| **三极管噪声源** | JDQ49A 鼓机 | BE 结反向击穿 | 省专用噪声 IC |

## 6. 三条路线的"模数分界线"对比

```
JDQ49A:     [数字分频(三极管)] | [滤波+包络+VCA+鼓机(三极管+RC)]
                              ↑ 分界在三极管"开关态 vs 线性态"

3786:        [分频+键选(CMOS 数字)] | [滤波+缓冲(CMOS 线性) + 包络(RC) + 加法(电阻)]
                              ↑ 分界在 CD4069UB 是否偏置线性区

本项目 v1.4: [全部数字: 分频+相位累加+包络表+ROM 查表+DAC]
                              ↑ 没有"模数分界线",全数字,只有 DAC 后才是模拟

未来混合版?: [分频+相位累加(74HC)] | [包络(电容) + 滤波(RC) + 混音(电阻)]
                              ↑ 把 vol 寄存器和包络表搬到模拟侧
```

## 7. 后续:SPICE 仿真切入点

LTspice 装好后,可以挑这些模块单独仿真验证:

1. **电容包络** —— RA×C / RR×C 充放电曲线,看 attack/release 时间是否符合钢琴感
2. **CD4046 PLL** —— VCO 控制电压 vs 输出频率,锁定范围
3. **CD4069UB 线性区** —— 偏置电阻选取,看线性范围
4. **加法合成节点** —— 多个方波叠加的频谱(FFT)
5. **模拟 VCA** —— OTA(LM13700)受 V_env 控制的增益曲线
6. **三极管 flip-flop**(JDQ49A 风格) —— 2 个 NPN 搭的 ÷2 分频器,验证触发灵敏度
7. **三极管噪声源** —— BE 结反向击穿产生白噪声,看频谱

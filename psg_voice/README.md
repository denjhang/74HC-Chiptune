## 共享资源
- **控制器**: FT232H (CJMCU 板, D2XX + MPSSE), 12 根线
  - C0-C7 = 数据总线, D4=LE, D5=A0, D6=RST, D7=A1 (v0.3+)
  - v0.4 起改 YM2413 风格: A0/CS/WR 地址数据复用
- **时钟**: 外部有源晶振 (方波/噪音 64kHz 预分频甜点, 波形通道 4MHz 直连)
- **电平转换**: LS373 (FT232H 3.3V → 74HC 5V, v0.1 验证必需, A0 也要过)
- **DAC**: TLC7524 (正向接法: REF=信号输入, OUT1=输出, RFB=GND)
- **输出**: OUT1 → 耦合电容 → 运放 → 喇叭

## 演进脉络
- **v0.1**: 单通道方波直驱. 验证 74HC PSG 核心 + FT232H 控制 + 电平转换 + HCT74 高频.
- **v0.2**: + 4-bit 音量 (TLC7524 衰减器). 核心不动, 加 HC374 + TLC7524.
- **v0.3 (PSG2)**: + 噪音通道 (LFSR). 用户创新: HC00+电容模拟噪声注入 D[0], 永不锁死.
- **v0.4 (PSG3)**: YM2413 总线 (地址/数据复用, 8 寄存器). + 简谱宏解析播放器 (ini 音级+乐器块+双轨).
- **v0.5 (PSG3)**: + 波形通道. CD4029 失败 → HC161 Q→DAC 失败 → rev.a 分频链锯齿 ✅ → rev.b 三角 ❌ → rev.c 调音色 → rev.d/d2 ROM 查表 → **rev.e2 双模式 (波形+PCM)**.

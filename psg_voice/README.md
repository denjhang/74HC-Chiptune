# psg_voice — 单音/多通道 PSG 系列

用 74HC 逻辑 + FT232H 控制的方波合成器。按版本演进，每版对应一张 EasyEDA 原理图。

## 版本

| 版本 | 状态 | 内容 |
|------|------|------|
| [PSG1 v0.1](PSG1%20v0.1/) | ✅ 已完成 | 单通道方波，5 片 74HC + LS373 电平转换 + FT232H，实测发声 |
| [PSG1 v0.2](PSG1%20v0.2/) | 🚧 规划中 | **v0.1 + 4-bit 音量**（TLC7524 衰减器），单通道不变 |
| PSG2 v0.3+ | 📋 待定 | 扩展到多通道（音量验证成功后再开）|

## 共享资源
- 控制器：FT232H（D2XX + MPSSE，详见各版 docs/mcu-interface.md）
- 时钟：外部时钟模块（v0.1 用 64kHz 甜点）
- 电平转换：LS373（FT232H 3.3V → 74HC 5V，v0.1 验证必需）

## 演进脉络
- **v0.1**：单通道，直驱（period 走 C0-C7，控制走 D4-D6）。验证了 74HC PSG 核心、FT232H 控制、电平转换、HCT74 高频。
- **v0.2**：v0.1 基础上加 4-bit 音量（TLC7524 衰减器）。核心电路不动，只加 1 片 HC374（音量锁存）+ 1 片 TLC7524。验证音量调节可行性。
- **v0.3+（PSG2）**：音量验证成功后扩展到多通道，届时引入 HC138 地址译码 + 总线复用。

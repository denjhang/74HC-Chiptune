# PSG1 v0.2 接线表

> **v0.1 + 4-bit 音量（TLC7524 衰减器）**，去掉 gate，toggle_q 直接经 TLC7524 衰减输出。
> 基于 v0.1 接线表，标注 **改动点**（加 HC374 + TLC7524，去 HC00 的 gate 与门）。

## 芯片位置

| 位号 | 型号 | 功能 | vs v0.1 |
|------|------|------|---------|
| U0 | 74LS373 | 电平转换层（FT232H 3.3V→5V）| 复用 |
| U1 | 74HC373 | period 锁存 | 复用 |
| U2 | 74HC161 | 计数器低 4 位 | 复用 |
| U3 | 74HC161 | 计数器高 4 位 | 复用 |
| U4 | 74HC00 | PE 反相（仅第 1 路，**gate 与门拆除**）| **改动：去 gate** |
| U5 | 74HCT74 | toggle 翻转 | 复用 |
| **U6** | **74HC374** | **音量码锁存（D0-D3，A0 选通）**| **新增** |
| **U7** | **TLC7524** | **音量衰减器（反向电压模式）**| **新增** |
| U8 | 喇叭 | 接 TLC7524 REF 输出 | 改接 U7（原接 U4） |

## 总线信号（去 gate，加 A0）

| 信号 | 来源 | 去向 | vs v0.1 |
|------|------|------|---------|
| `clk` | 外部时钟 64kHz | U2.P2, U3.P2, U5.P3 | 不变 |
| `rst_n` | FT232H D6 (经 LS373) | U2.P1, U3.P1, U5.P1, U5.P13 | 不变 |
| `period_le` | FT232H D4 (经 LS373) | U1.P11（HC373 LE）| 不变 |
| **`A0`** | **FT232H D5 (经 LS373)** | **U6.P11（HC374 CP）**| **v0.1 是 gate，现改 A0** |
| `period_in[7:0]` | FT232H C0-C7 (经 LS373) | U1.D0-D7 | 不变 |
| `toggle_q` | U5.P9 | **U7.P1（TLC7524 OUT1）** | **改：原接喇叭→接 TLC7524** |
| `vol[3:0]` | U6.Q0-Q3 | U7.DB4-DB7 | 新增 |

> **gate 去除**：v0.1 的 HC00 第 2/3 路（gate 与门）拆除。toggle_q 不再经 gate，直接输出。
> **静音方式**：写 vol=0000 → TLC7524 输出 0 → 静音（替代 gate）。

## U6 — 74HC374（音量码锁存，新增）

8 位 D 触发器，CP 上升沿锁存。只用了低 4 位（D0-D3），高 4 位接地。

| Pin | 信号 | 连接 |
|-----|------|------|
| 1 | /OE | → GND（常输出）|
| 11 | CP | ← **A0**（FT232H D5 经 LS373，写脉冲）|
| 9 | D0 | ← FT232H D0（经 LS373，音量 bit0）|
| 8 | D1 | ← FT232H D1（经 LS373，音量 bit1）|
| 7 | D2 | ← FT232H D2（经 LS373，音量 bit2）|
| 6 | D3 | ← FT232H D3（经 LS373，音量 bit3）|
| 12-15 | D4-D7 | → GND（不用，音量只用 4 位）|
| 2 | Q0 | → U7.P7（TLC7524 DB4）|
| 3 | Q1 | → U7.P6（TLC7524 DB5）|
| 4 | Q2 | → U7.P5（TLC7524 DB6）|
| 5 | Q3 | → U7.P4（TLC7524 DB7）|
| 16-19 | Q4-Q7 | 悬空（不用）|
| 20 | VCC | +5V |
| 10 | GND | GND |

> **音量接 DB4-DB7 高位**：vol=1 时 DAC 输入=16（6% 幅度，可听见），分布合理。
> **写时序**：FT232H 把 vol 放 D0-D3，A0 给上升沿，HC374 锁存。

## U7 — TLC7524（音量衰减器，反向电压模式，新增）

乘法 DAC，反向电压模式（REF 当输出，OUT1 当输入），零外部元件。

| Pin | 信号 | 连接 | 说明 |
|-----|------|------|------|
| 1 | OUT1 | ← U5.P9（**toggle_q 方波**）| 参考输入（被衰减的信号）|
| 2 | OUT2 | → GND | 模拟地 |
| 3 | GND | → GND | |
| 4 | DB7 | ← U6.Q3（音量 bit3，MSB）| |
| 5 | DB6 | ← U6.Q2（音量 bit2）| |
| 6 | DB5 | ← U6.Q1（音量 bit1）| |
| 7 | DB4 | ← U6.Q0（音量 bit0）| |
| 8 | DB3 | → GND | 固定 0 |
| 11 | DB2 | → GND | 固定 0 |
| 12 | DB1 | → GND | 固定 0 |
| 13 | DB0 | → GND | 固定 0 |
| 14 | WR | → GND | 常写入（直通模式，音量码实时跟随）|
| 15 | CS | → GND | 常选通 |
| 9 | VDD | → +5V | |
| 10 | GND | → GND | |
| **16** | **REF** | **→ 喇叭** | **衰减后的方波输出** |

> **反向电压模式公式**：VO = VI × (D/256)
> - toggle_q（0/5V 方波）进 OUT1，REF 输出 = 5V × (D/256) 的方波
> - D = vol<<4（vol 0-15 → D = 0-240）
> - vol=15 → D=240 → 输出 4.7V 方波（满音量）
> - vol=0 → D=0 → 输出 0V（静音）

## U4 改动（HC00 去 gate）

v0.1 的 HC00 第 2/3 路（gate 与门）拆除，仅保留第 1 路（PE 反相）：

| Pin | v0.1 | v0.2 |
|-----|------|------|
| 4 (A2) | ← gate | → GND（不用）|
| 5 (B2) | ← toggle_q | 悬空 |
| 6 (Y2) | → gate_nand1 | 悬空 |
| 8 (Y3) | → wave_out → 喇叭 | **悬空**（不再经 gate）|
| 9,10 (A3,B3) | ← gate_nand1 | → GND |

> **toggle_q 直接输出**：U5.P9（toggle_q）不再经 HC00 的 gate 与门，直接接 U7.P1（TLC7524 OUT1）。
> HC00 第 1 路（PE 反相，Pin 1/2/3）保持不变。

## 4-bit 音量档位

vol[3:0] → D = vol << 4 → 输出幅度 = 5V × D/256

| vol | D 值 | 输出幅度 | 听感 |
|-----|------|---------|------|
| 0000 | 0 | 0V | 静音 |
| 0001 | 16 | 0.31V | 极小 |
| 0100 | 64 | 1.25V | 小声 |
| 1000 | 128 | 2.5V | 中声 |
| 1100 | 192 | 3.75V | 大声 |
| 1111 | 240 | 4.69V | 满音量 |

## 写时序总表

| 操作 | FT232H 动作 |
|------|------------|
| 写 period | D0-D7=period（C口），D4(LE) 上升沿 → HC373 锁存 |
| 写音量 | D0-D3=vol（C口低4位），D5(A0) 上升沿 → HC374 锁存 |
| 静音 | 写 vol=0000 |
| 复位 | D6(RST) 拉低再拉高 |

## 电源（所有芯片）

| 芯片 | VDD | GND |
|------|-----|-----|
| 74LS373 (U0) | Pin 16 | Pin 8 |
| 74HC373 (U1) | Pin 20 | Pin 10 |
| 74HC161 (U2,U3) | Pin 16 | Pin 8 |
| 74HC00 (U4) | Pin 14 | Pin 7 |
| 74HCT74 (U5) | Pin 14 | Pin 7 |
| **74HC374 (U6)** | **Pin 20** | **Pin 10** |
| **TLC7524 (U7)** | **Pin 9** | **Pin 3, 10** |

> 每个 VDD 就近加 0.1μF 去耦电容。所有 GND 共地。

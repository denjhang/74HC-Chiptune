# PSG3 v0.5 rev.e2 接线表 — 波形通道 (双模式: 波形 + PCM 采样)

> rev.e2 = rev.e 内容重排版. 位号从 U1 起, 全部 12 片连续编号, 无附号 (U27b/U0a 之类).
> 双模式: 16 种波形 (mode=0) / 16 个 PCM 采样槽 (mode=1).
> 采样: 8-bit PCM, 8K/槽 (8kHz × 1秒).
> ROM: SST39SF010 (128K, 全用). clk = 4MHz. reg3/4/5.

---

## 一、寄存器

```
reg3 (0x08): period12[7:0]
reg4 (0x10): period12[11:8]<<4 | vol[3:0]
reg5 (0x20):
  bit7: mode (0=波形, 1=采样)
  bit6: trig (1=清地址, 0=放手; 上位机写 1 再写 0)
  bit5-4: 预留
  bit3-0: sel[3:0] (选 16 波形 / 16 采样)
```

> reg5 位含义 (对应 Q 脚): mode=bit7=Q7, trig=bit6=Q6, sel[3:0]=bit3-0=Q3/Q2/Q1/Q0.
> reg4: vol[3:0]=bit3-0=Q3/Q2/Q1/Q0.

---

## 二、地址分配

```
A0-7:   U4/U5 (HC161×2), 两模式共用
A8-11:  U6 (HC161), 采样模式级联跑, 波形模式冻结 Q=0
A12:    U7 (HCT74 toggle), U6.TC 驱动
A13-16: reg5.sel[3:0], 两模式同线
```

波形模式: U6 冻结 → A8-12=0, 只用 A0-7 (256 点) + A13-16 (sel).
采样模式: U6 级联 → A8-12 连续跑, A0-12 = 8K 连续地址.

---

## 三、信号定义 (全局网络标签, 本文内同名 = 同一根线)

> 下文所有芯片表只写本表的网络标签名 + 物理来源/去向 (芯片号+脚).
> 同一标签名在本文内任意位置出现都指同一根物理线.

### 3.1 reg5 输出 (总线层 U7 的 Q 端, rev.e 重新定义位含义)

| 标签 | 来源 | 含义 |
|------|------|------|
| sel0 | reg5.Q0 (P2) | 选槽 bit0 |
| sel1 | reg5.Q1 (P5) | 选槽 bit1 |
| sel2 | reg5.Q2 (P6) | 选槽 bit2 |
| sel3 | reg5.Q3 (P9) | 选槽 bit3 |
| — (未用) | reg5.Q4 (P12) | bit4 预留, 悬空 (不接任何负载) |
| — (未用) | reg5.Q5 (P15) | bit5 预留, 悬空 (不接任何负载) |
| trig | reg5.Q6 (P16) | 触发脉冲 (上位机写 1 再写 0) |
| mode | reg5.Q7 (P19) | 0=波形模式 / 1=采样模式 |

> reg5 全 8 个 Q 脚均有交代: Q0-Q3=sel, Q4/Q5=预留悬空, Q6=trig, Q7=mode.
> 预留脚悬空 (HC374 Q 端为推挽输出, 悬空无危害, 便于后续扩展接线).

### 3.2 reg4 输出 (音量)

| 标签 | 来源 | 含义 |
|------|------|------|
| vol0 | reg4.Q0 (P2) | 音量 bit0 (LSB) |
| vol1 | reg4.Q1 (P5) | 音量 bit1 |
| vol2 | reg4.Q2 (P6) | 音量 bit2 |
| vol3 | reg4.Q3 (P9) | 音量 bit3 (MSB) |

### 3.3 公共信号 (复位/时钟)

| 标签 | 来源 | 含义 |
|------|------|------|
| rst_n | 总线层 U0b.Q2 | 复位 (低有效), 来自总线层电平转换 |
| clk | 晶振 X1 | 4MHz 主时钟 |

### 3.4 period 计数器 (直出, 无组合)

| 标签 | 含义 |
|------|------|
| freq_tc | period12 计满一周期进位 (高有效脉冲), U3.TC |

### 3.5 地址计数器 TC 级联链 (HC161 进位, 直出)

| 标签 | 含义 |
|------|------|
| tc_a03 | 地址 A0-3 计满 16 进位, U4.TC |
| tc_a47 | 地址 A0-7 计满 256 进位, U5.TC |
| tc_a811 | 地址 A0-11 计满 4096 进位, U6.TC |

### 3.6 地址位 (HC161 Q / HCT74 Q)

| 标签 | 含义 |
|------|------|
| a0 .. a11 | 地址线 A0-A11 (U4/U5/U6 的 Q 端, 从低到高) |
| a12 | 地址线 A12 (U7 FF1 每 4096 步翻转一次) |

### 3.7 ROM 数据线 (ROM → DAC, 直连)

| 标签 | 含义 |
|------|------|
| d0 .. d7 | U10 ROM 输出 D0-D7, 直连 U11 TLC7524 DB0-DB7 (d0=LSB, d7=MSB) |

### 3.8 组合信号 (经 U8 HC08 / U9 HC04 产生)

| 标签 | 产生门 | 定义 |
|------|--------|------|
| tc_pcm | U8 门1 | tc_a47 AND mode (采样模式才把进位传给 U6) |
| trig_clr | U8 门2 | trig AND mode (采样模式 trig=1 时请求清地址) |
| at_8191 | U8 门3 | tc_a811 AND a12 (地址走到 8191) |
| step | U8 门4 | freq_tc AND n_at8191 (地址步进使能, 高有效) |
| n_trig_clr | U9 门1 | ~trig_clr (给 HC161 的 ~PE, 低有效预置) |
| n_at8191 | U9 门2 | ~at_8191 (给 U8 门4 输入 B) |

> HC161 的 ~PE 低有效: n_trig_clr=0 时预置 P0-3 (=0, 清地址).
> step 高有效: step=1 时 CEP/CET 使能, 地址前进一步. 到 8191 后 step=0, 地址冻结.

## 四、ROM 分区 (128K = 16 槽 × 8K, sel 选槽)

```
sel=0  (0x00000-0x01FFF): 波形 sine    / 采样 BD 底鼓
sel=1  (0x02000-0x03FFF): 波形 tri     / 采样 SD 军鼓
sel=2  (0x04000-0x05FFF): 波形 saw     / 采样 HH 踩镲
sel=3  (0x06000-0x07FFF): 波形 rsaw    / 采样 TOM 嗵鼓
sel=4  (0x08000-0x09FFF): 波形 square50/ 采样 RIM 边击
sel=5  (0x0A000-0x0BFFF): 波形 pulse25 / 采样 TOP 踩镲开
sel=6  (0x0C000-0x0DFFF): 波形 pulse12 / 采样 Piano 钢琴
sel=7  (0x0E000-0x0FFFF): 波形 wt.sin  / 采样 SlapBass 贝斯
sel=8  (0x10000-0x11FFF): 波形 wt.halfsin / 采样 Oboe 双簧管
sel=9  (0x12000-0x13FFF): 波形 wt.abssin  / 采样 Trumpet 小号
sel=10 (0x14000-0x15FFF): 波形 pacman.sine / 采样 Strings 弦乐
sel=11 (0x16000-0x17FFF): 波形 pacman.tri  / 采样 Harp 竖琴
sel=12 (0x18000-0x19FFF): 波形 pacman.saw  / 采样 Guitar 吉他
sel=13 (0x1A000-0x1BFFF): 波形 pacman.step4/ 采样 Shakuhachi 尺八
sel=14 (0x1C000-0x1DFFF): 波形 pulse75    / 采样 Blow 吹管
sel=15 (0x1E000-0x1FFFF): 波形 pulse37    / 采样 Oboe2
```

波形模式只读每槽前 256 字节 (A8-12=0). 采样模式读满 8K (A0-12 跑). 两模式 mode 互斥不冲突.

---

## 五、采样规格

- 8-bit PCM (0-255, 128=静音中心)
- 每槽 8K = 8192 样本 = 8kHz × 1.024 秒
- period12 控制回放速度 (= 音高). 默认 8kHz (period12=3596)
- 采样首尾 crossfade (autoloop.py), 单次播放到 8191 自动停 (step=0)

### U1-U3 — HC161×3 period12 三级联 (同 rev.d, 不变)

> 接线完全同 rev.d 第四节 (总线层 U20-U22). 这里只标注 rev.e2 用到的输出.
> U1/U2 的 Q 全悬空 (只内部 TC 级联用). U3.TC 输出 = freq_tc.

**U1 (period12[3:0])** — MR←rst_n, CP←clk, P0-3←reg3.Q0-3, CEP/CET→+5V, ~PE←(rev.d 接 ~tc_hi, 本表不涉及), TC→U2.CEP/CET, Q 悬空.

**U2 (period12[7:4])** — MR←rst_n, CP←clk, P0-3←reg3.Q4-7, CEP/CET←U1.TC, TC→U3.CEP/CET, Q 悬空.

**U3 (period12[11:8]) 输出**:

| Pin | 来源 | 连接到 | 网络标签 | 说明 |
|-----|------|--------|----------|------|
| 11 (Q3) | — | 悬空 | — | period 高位, 本表不读 Q (只内部 TC 级联) |
| 12 (Q2) | — | 悬空 | — | |
| 13 (Q1) | — | 悬空 | — | |
| 14 (Q0) | — | 悬空 | — | |
| 15 (TC) | — (本芯片) | U8.A4 (P12) | freq_tc | period 一周期进位脉冲 |

> freq_tc 不经整形, 直接进 U8 门4 做 step 的 AND 输入 (rev.e 架构: period 进位驱动地址步进).

## 九、关键信号流 (用网络标签, 不写芯片号)

```
clk ──→ period 计数器 (U1-U3) → freq_tc
                                    │
                                    ▼ (进 U8 门4)
freq_tc ──AND── n_at8191 ──→ step ──→ U4/U5 CEP/CET (地址步进)
                                    ↘ (U6 不接 step, 接 tc_pcm)

地址计数器级联链:
  tc_a03 (U4 计满16) ──→ U5.CEP/CET
  tc_a47 (U5 计满256) ──→ U8 门1 ──AND── mode ──→ tc_pcm ──→ U6.CEP/CET
  tc_a811 (U6 计满4096) ──→ U7.CLK1 (a12 toggle)
                          ↘──→ U8 门3 ──AND── a12 ──→ at_8191

停止逻辑:
  at_8191 ──→ U9 门2 ──→ n_at8191 ──→ U8 门4 (拉低 step) → 地址冻结

清地址 (触发):
  trig ──→ U8 门2 ──AND── mode ──→ trig_clr ──→ U9 门1 ──→ n_trig_clr ──→ U4/U5/U6 ~PE

ROM 地址:  a0-a12 (地址计数器) + sel0-sel3 (reg5) ──→ U10.A0-A16
ROM 数据:  d0-d7 ──→ U11.DB0-7 ──→ U11.OUT1 ──→ U12.REF ──→ U12.OUT1 ──→ 喇叭
```

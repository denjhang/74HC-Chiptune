# PSG3 v0.5 接线表 — 接口层（YM2413 总线）

> 接口层完全独立，不借门、不复用通道层任何芯片。
> 物理接线与 v0.4 完全相同（U0-U13），只有 U2-U9 寄存器的 Q 去向按三通道重新分配。
> 方波/波形/噪音通道接线表各自独立，见 `wiring-table-square.md` / `wiring-table-uni.md` / `wiring-table-noise.md`。

---

## 一、FT232H 接线（12 根线，物理不动，语义改变）

| FT232H | PSG3 信号 | 用途 |
|--------|----------|------|
| C0-C7 | D0-D7 复用总线 | 地址 + 数据复用 |
| D4 | A0 | 0=写地址, 1=写数据 |
| D5 | /WR | 写脉冲（上升沿锁存）|
| D6 | /RST | 复位（低有效）|
| D7 | /CS | 片选（事务边界，必须控制，不接 GND）|

> ⚠️ 所有 FT232H 信号经 LS373 电平转换（3.3V→5V）。
> ⚠️ /CS 不接 GND——事务保护壳，消除"短暂地址值"中间态。

## 二、芯片清单（接口层独立，13 片）

| 位号 | 型号 | 功能 |
|------|------|------|
| U0a | 74LS373 | 电平转换：8 位数据总线 |
| U0b | 74LS373 | 电平转换：4 位控制（A0/WR/RST/CS）|
| U1 | 74HC374 | 地址锁存（边沿，独热码 8 位）|
| U2-U9 | 74HC374 ×8 | 数据寄存器 reg0-reg7（reg0/1=方波, reg2=噪音, reg3/4/5=波形, reg6/7 预留）|
| U10 | 74HC04 | NOT(/CS)、NOT(A0) |
| U11 | 74HC08 | addr_cp + data_strobe 组合 |
| U12 | 74HC08 | reg_cp[0-3] 选通 |
| U13 | 74HC08 | reg_cp[4-7] 选通 |

## 三、地址映射（独热码，每位选通一个 reg）

| 寄存器 | 独热码 | ADDR 位 | 芯片 | 用途 |
|--------|--------|---------|------|------|
| reg0 | 0x01 | ADDR[0] | U2 | 方波 period（8 bit, v0.4 不变）|
| reg1 | 0x02 | ADDR[1] | U3 | 方波控制（vol/duty/mode, v0.4 不变）|
| reg2 | 0x04 | ADDR[2] | U4 | 噪音控制（vol/freq/bind）|
| reg3 | 0x08 | ADDR[3] | U5 | **波形 period[7:0]**（v0.5 新）|
| reg4 | 0x10 | ADDR[4] | U6 | **波形 period[11:8] \| vol**（v0.5 新）|
| reg5 | 0x20 | ADDR[5] | U7 | **波形 duty \| wave \| mode**（v0.5 新）|
| reg6 | 0x40 | ADDR[6] | U8 | 预留 |
| reg7 | 0x80 | ADDR[7] | U9 | 预留 |

---

## 四、选通逻辑

```
cs_active   = NOT(/CS)
a0_n        = NOT(A0)
addr_cp     = cs_active AND a0_n AND /WR     // 写地址: /WR↑ + A0=0 → 锁地址
data_strobe = cs_active AND A0     AND /WR   // 写数据: /WR↑ + A0=1 → 锁数据
reg_cp[n]   = ADDR[n] AND data_strobe         // 独热码地址一对一选通
```

## 五、U0a — 74LS373（电平转换：8 位数据总线）

> ⚠️ **LS373 与 HC374 引脚不同**（虽然都是 DIP-20）：
> - LS373: D0-D7=P3,4,**6**,8,13,14,**16**,18；Q0-Q7=P2,5,**7**,9,12,15,**17**,19
> - HC374: D0-D7=P3,4,**7**,8,13,14,**17**,18；Q0-Q7=P2,5,**6**,9,12,15,**16**,19

| D0-D7 (P3,4,6,8,13,14,16,18) | ← FT232H C0-C7（3.3V）| 数据输入 |
| LE (P11) | → VCC（常透明）| 纯缓冲 |
| /OE (P1) | → GND | 常输出 |
| Q0-Q7 (P2,5,7,9,12,15,17,19) | → 总线 D0-D7（5V）| → U1.D, U2-U9.D |
| VCC (P20) | → +5V | |
| GND (P10) | → GND | |

## 十三、与通道层的接口（信号交接）

接口层向通道层输出的信号：

| 信号 | 来源 | 去向 |
|------|------|------|
| reg0.Q[7:0] | U2 | → 方波通道 period |
| reg1.Q[7:0] | U3 | → 方波通道 vol(bit0-3)/duty(bit4-5)/mode(bit6)/ref(bit7) |
| reg2.Q[7:0] | U4 | → 噪音通道 vol(bit0-3)/freq(bit4-5)/bind(bit6) |
| reg3.Q[7:0] | U5 | → 波形通道 period12[7:0] |
| reg4.Q[7:0] | U6 | → 波形通道 period12[11:8](bit4-7)/vol(bit0-3) |
| reg5.Q[7:0] | U7 | → 波形通道 duty(bit4-7)/wave(bit2-3)/mode(bit1)/预留(bit0) |
| rst_n | U0b.Q2 | → 通道层所有 MR/CLR |
| sq_tc（方波→噪音绑定）| 方波通道 HC161 TC | → 噪音通道 HC00（绑定模式时钟源）|

> sq_tc 是方波通道向噪音通道的跨通道信号（绑定模式用）。
> ⚠️ **必须经 HC00 反相器隔离**（2026-07-08 上板发现）：直连会反向干扰方波。详见噪音通道接线表。
> v0.5 新增: reg3/4/5 → 波形通道 (三通道并存, 各自独立 TLC7524 输出)。

## 十四、写时序（FT232H 上位机）

```
写 reg[n] = data（完整事务）:

/CS = 0                           // 事务开始
bus = (1<<n); A0=0; /WR ↓→↑       // 第 1 拍: 写地址
bus = data;   A0=1; /WR ↓→↑       // 第 2 拍: 写数据
/CS = 1                           // 事务结束
```

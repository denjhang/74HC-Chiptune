# PSG3 v0.5 rev.e3 接线表 — PCM 采样通道 (无限循环, 最简验证版)

> rev.e3 = rev.e2 砍掉所有组合逻辑 (HC08/HC04) 和 7474 toggle, PCM 专用最简版.
> 目的: 隔离 rev.e2 的故障 (7474/组合逻辑接错), 先验证 PCM 基本播放.
> **无限循环**: 地址 0-8191 跑完自动回绕 (HC161 自由计数), 不停, 无触发, 无停止逻辑.
> **频率可调**: period12 控制回放步进速率 (= 采样频率).

---

## 一、与 rev.e2 的区别

| 项 | rev.e2 | rev.e3 |
|----|--------|--------|
| 模式 | 双模式 (波形+PCM) | **PCM only** |
| a12 | U7 7474 toggle (U6.TC 驱动) | **U7 HC161 Q0** (U6.TC 级联, 无 toggle) |
| 组合逻辑 | U8 HC08 + U9 HC04 (6 信号) | **全砍** |
| 触发/停止 | trig_clr / at_8191 / step | **全砍** (无限循环, 软件不控制启停) |
| 地址计数 | U4-U6 (12 位) | **U4-U7 (13 位, 加 1 片 HC161)** |
| reg5 | mode/trig/预留/sel | **sel only** (Q0-3), Q4-7 悬空 |
| 芯片数 | 12 | **10** |

> rev.e3 是 rev.e2 的子集 + 1 片 HC161 (a12), 砍掉 7474/HC08/HC04 共 3 片, 净 -3 片.

---

## 二、寄存器

```
reg3 (0x08): period12[7:0]        ← 步进速率 (采样频率)
reg4 (0x10): period12[11:8]<<4 | vol[3:0]
reg5 (0x20):
  bit7-4: 悬空 (不用)
  bit3-0: sel[3:0] (选 16 个 PCM 槽, → ROM.A13-16)
```

> reg5 只有 sel, 无 mode/trig. 上位机写 sel 选槽, 写 period12 调速率, 写 vol 调音量.
> 无启停控制: 上电就开始循环播放 sel 选中的槽, 换 sel 即换采样.

## 四、ROM 分区 (128K = 16 槽 × 8K, PCM only)

```
sel=0  (0x00000-0x01FFF): BD 底鼓
sel=1  (0x02000-0x03FFF): SD 军鼓
sel=2  (0x04000-0x05FFF): HH 踩镲
sel=3  (0x06000-0x07FFF): TOM 嗵鼓
sel=4  (0x08000-0x09FFF): RIM 边击
sel=5  (0x0A000-0x0BFFF): TOP 踩镲开
sel=6  (0x0C000-0x0DFFF): Piano 钢琴
sel=7  (0x0E000-0x0FFFF): SlapBass 贝斯
sel=8  (0x10000-0x11FFF): Oboe 双簧管
sel=9  (0x12000-0x13FFF): Trumpet 小号
sel=10 (0x14000-0x15FFF): Strings 弦乐
sel=11 (0x16000-0x17FFF): Harp 竖琴
sel=12 (0x18000-0x19FFF): Guitar 吉他
sel=13 (0x1A000-0x1BFFF): Shakuhachi 尺八
sel=14 (0x1C000-0x1DFFF): Blow 吹管
sel=15 (0x1E000-0x1FFFF): Oboe2
```

每槽 8192 样本, 无限循环 (0→8191→0→...). 采样频率 = 4MHz / (8192 × (4096 - period12)).
默认 8kHz: period12 = 4096 - 4MHz/(8192×8000) = 4096 - 61 = 4035.

## 五、芯片清单 (10 片, 位号从 U1 起)

> 本文位号独立. U1-U3 = period 计数器 (= 总线层 U20-U22).
> U4-U7 = 地址计数器 (13 位, a0-a12). U8 = ROM. U9/U10 = DAC.

| 位号 | 型号 | 功能 |
|------|------|------|
| U1 | HC161 | period12[3:0] |
| U2 | HC161 | period12[7:4] |
| U3 | HC161 | period12[11:8], TC → freq_tc |
| U4 | HC161 | 地址 a0-a3 (低 4 位) |
| U5 | HC161 | 地址 a4-a7 (中 4 位) |
| U6 | HC161 | 地址 a8-a11 (高 4 位) |
| U7 | HC161 | 地址 a12 (只用 Q0, U6.TC 级联) |
| U8 | SST39SF010 | ROM (128K) |
| U9 | TLC7524 | PCM DAC (CS=clk) |
| U10 | TLC7524 | 音量衰减 |

> vs rev.e2 (12 片): 砍 7474/HC08/HC04 (-3), 加 a12 的 HC161 (+1), 净 -2 片.
> rev.e3 无组合逻辑, 无 toggle, 接线最简, 适合先验证 PCM.

## 八、关键信号流

```
clk ──→ period 计数器 (U1-U3) → freq_tc
                                    │
                                    ▼
U4 CEP/CET ← freq_tc (每 period 走一步)
U4 → tc_a03 → U5 CEP/CET
U5 → tc_a47 → U6 CEP/CET
U6 → tc_a811 → U7 CEP/CET (a12)
    │ a0-a12 (13 位地址, 0-8191 无限循环)
    ▼
U8 ROM (A0-12=地址, A13-16=sel) → d0-7 → U9 DAC (CS=clk) → U10 (vol) → 喇叭

无组合逻辑, 无 toggle, 无触发, 无停止. 上电即循环播放.
```

## 九、采样频率公式

```
采样频率 = 4MHz / (8192 × (4096 - period12))
period12 = 4096 - 4MHz / (8192 × 采样频率)
```

| 采样频率 | period12 | 说明 |
|----------|----------|------|
| 8kHz (默认) | 4035 | 标准 PCM 速率 |
| 4kHz | 3974 | 慢一半 (降调) |
| 16kHz | 4065 | 快一倍 (升调) |
| 1kHz | 3606 | 很慢 |
| 488Hz | 3095 | 鼓组低频 |

> period12 越大, 采样频率越高. 8192 步/槽, 无限循环.

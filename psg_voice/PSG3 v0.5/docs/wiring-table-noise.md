# PSG3 v0.5 接线表 — 噪音通道

> **照抄 PSG3 v0.4 噪音通道 rev.a（已上板验证）。**
> v0.5 变化: reg2→reg3(0x08), clk 改 sq_clk (÷63 预分频).
> 绑定时钟源改 tc_out (统一通道 freq_tc).

---

## 一、芯片清单 (7 片 + 2 电容, 同 v0.4)

| 位号 | 型号 | 功能 |
|------|------|------|
| U40 | CD4070 | LFSR 反馈 Q7⊕Q5⊕Q4⊕Q3 (门1-3) |
| U41 | 74HC374 | LFSR 主体 (8 位 D 触发器阵列) |
| U42 | 74HC161 | 独立分频 (sq_clk ÷2/÷4/÷8/÷16) |
| U43 | 74HC153 | 频率选通 4 选 1 |
| U44 | TLC7524 | 噪音衰减器 (REF=Q7) |
| U45 | 74HC00 | 绑定 2 选 1 (4 门用满) |
| U46 | 74HC00 | 防锁死振荡器 (门3+电容, 独立芯片) |
| C1, C2 | 0.1μF ×2 | U46 门3 张弛振荡器电容 |

---

## 二、寄存器 (reg3 = 0x08)

```
reg3 (0x08): 噪音控制
  bit0-3: 音量 (4-bit)
  bit4-5: 频率挡 (00=÷2 / 01=÷4 / 10=÷8 / 11=÷16)
  bit6:   绑定开关 (0=独立 / 1=绑定统一通道 freq_tc)
  bit7:   预留
```

---

## 三、关键改动 (vs v0.4)

| 项 | v0.4 | v0.5 |
|----|------|------|
| 寄存器地址 | reg2 (0x04) | **reg3 (0x08)** |
| 独立分频 clk | 64kHz 直连 | **sq_clk (÷63, 63492Hz)** |
| 绑定时钟源 | 方波 tc_hi | **统一通道 tc_out (freq_tc)** |

> 完整逐脚表照抄 v0.4 rev.a (`psg_voice/PSG3 v0.4/docs/wiring-table-noise.md`).
> 改动: U42.CP 从 clk 改 sq_clk; 绑定输入从 tc_hi 改 tc_out.

---

|------|------|------|
| nz_audio | U44.OUT1 (TLC7524) | → 耦合电容 → 运放 → 喇叭 (混音) |

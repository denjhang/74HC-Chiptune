# PSG3 v0.5 接线表 — 预分频模块 (4MHz → ÷63 → sq_clk)

> v0.5: 晶振 4MHz. 统一通道吃全速 4MHz (16步@4M).
> 噪音通道需 ~64kHz 等效, 经 ÷63 预分频 → sq_clk = 63492Hz (-0.8%, 软件补偿).

---

## 一、为什么 ÷63

```
4MHz ÷ 63 = 63492Hz (偏离 64kHz -0.8%, 半音以内, 软件补偿)
reload = 256 - 63 = 193 (0xC1)
```

| ÷N | 输出 | 偏离 64k |
|----|------|---------|
| 62 | 64516 | +0.8% |
| **63** | **63492** | **-0.8% ✓** |
| 64 | 62500 | -2.3% |

软件补偿: `freq_to_period()` 里 `64000 → 63492` (1 行改动).

---

## 二、芯片选型: HC161 不是 7474

- ÷63 需要 6 位以上分频 (63 > 16). **7474 双 D 触发器最多 ÷4** (2 级 toggle), 不够.
- **HC161 reload 机制** (TC→PE 预置) 跟方波通道同款, 维护一致.
- 2 片 HC161 级联 (8-bit), reload=193, 周期=256-193=63.

---

## 三、芯片清单 (3 片)

| 位号 | 型号 | 功能 |
|------|------|------|
| U38a | 74HC161 | ÷63 低 4 位 (reload=193 低 nibble=0x1=0001) |
| U38b | 74HC161 | ÷63 高 4 位 (reload=193 高 nibble=0xC=1100, TC=sq_clk) |
| U39 | 74HC00 | PE 反相 (高位 TC → PE, 1 门, 余 3 门备用) |

> HC161 级联 + TC→PE reload 机制 (跟方波 period 计数器同款):
> 低位 CEP/CET=1 常走, 高位 CEP/CET=低位 TC. 高位 TC=1 时 PE 预置两片回 reload.
> 周期 = 256 - reload = 63 个 clk.

---

## 四、信号流

```
clk(4MHz) ──→ U38a HC161 (低位, reload=0x1)
                      │ TC (低位计满, 使能高位)
                      ▼
                U38b HC161 (高位, reload=0xC)
                      │ TC = sq_clk (÷63 输出, 每 63 clk 一个脉冲)
                      │   ↓ U39 HC00 反相 → PE
                      └──→ PE (两片共用, 高位 TC 时预置回 reload)
                      │
                      ▼ sq_clk (63492Hz)
                → 方波通道 CH0 (u_cnt_lo/hi, toggle)
                → 噪音通道 CH1 (u_div161 独立分频)
```

> ⚠️ sq_clk 是脉冲信号 (高位 TC, 每 63 clk 高一拍). 方波/噪音通道的 HC161 用 sq_clk 当 CP,
> 在 sq_clk 上升沿计数. 跟原来 64kHz 连续时钟等效 (每 63 个 4MHz clk 走一步).

---

## 五、U38a — 74HC161 (÷63 低位)

reload = 193 的低 4 位 = 0x1 = 0001 → D0=1, D1=D2=D3=0

| Pin | 信号 | 连接 |
|-----|------|------|
| 1 (MR) | ← rst_n | 复位 |
| 2 (CP) | ← **clk (4MHz)** | 计数时钟 |
| 3 (D0) | → +5V | reload bit0=1 |
| 4 (D1) | → GND | reload bit1=0 |
| 5 (D2) | → GND | reload bit2=0 |
| 14 (D3) | → GND | reload bit3=0 |
| 7 (CEP), 10 (CET) | → +5V | 常使能 |
| 9 (PE) | ← U39.Y1 (pe_n) | 计满预置 |
| 6,11,12,13 (Q0-3) | 悬空 | (只内部 TC 用) |
| 15 (TC) | → U38b.CEP, U38b.CET | **级联高位** |
| 16 VDD, 8 GND | | |

---

## 六、U38b — 74HC161 (÷63 高位)

reload = 193 的高 4 位 = 0xC = 1100 → D0=D1=0, D2=D3=1

| Pin | 信号 | 连接 |
|-----|------|------|
| 1 (MR) | ← rst_n | |
| 2 (CP) | ← clk | |
| 3 (D0) | → GND | reload bit4=0 |
| 4 (D1) | → GND | reload bit5=0 |
| 5 (D2) | → +5V | reload bit6=1 |
| 14 (D3) | → +5V | reload bit7=1 |
| 7 (CEP), 10 (CET) | ← **U38a.TC** | 级联使能 |
| 9 (PE) | ← U39.Y1 (pe_n) | 共用预置 |
| 6,11,12,13 (Q0-3) | 悬空 | |
| 15 (TC) | → **sq_clk** (方波/噪音时钟) + U39.A1 | **÷63 输出** |
| 16 VDD, 8 GND | | |

---

## 七、U39 — 74HC00 (PE 反相)

| 门 | 功能 | Pin |
|----|------|-----|
| 门1 | NOT(sq_clk) → pe_n | U38b.TC 短接 → 反相 → U38a/U38b.PE |
| 门2,3,4 | 闲置 | 接 GND |

---

## 八、电源

| 芯片 | VDD | GND |
|------|-----|-----|
| HC161 (U38a, U38b) | P16 | P8 |
| HC00 (U39) | P14 | P7 |

> 每片 VDD 就近加 0.1μF 去耦. 所有 GND 共地.

---

## 九、与 v0.4 的差异

| v0.4 (64kHz) | v0.5 (4MHz) |
|--------------|-----------------|
| 方波/噪音 CLK 直连 64kHz 晶振 | CLK ← sq_clk (经 ÷63 预分频) |
| 无预分频模块 | +3 片 (U38a/U38b/U39) |
| 晶振 64kHz | 晶振 4MHz |

> 方波/噪音通道内部逻辑 (HC161 period 计数/toggle/LFSR/HC153 等) **完全不变**,
> 只是 clk 来源从直连晶振改成 sq_clk. 详见 `wiring-table-square.md` / `wiring-table-noise.md`.

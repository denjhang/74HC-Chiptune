# PSG2 v0.3 噪音通道接线表 rev.a (HC374 版)

> **rev.a vs 原版 (wiring-table.md 第十一节) 的区别**:
> - LFSR 主体: HC164 → **HC374** (国产 HC164 不稳定 + 上电全 0 死锁)
> - 反馈: CD4070 (不变, 与 HC86 功能等效)
> - 启动逻辑: **RC 上电脉冲 + 借方波 U4 HC00 门 2/3/4** (0 额外芯片)
> - 芯片数: 7 片 (持平)
>
> rev.a 只改噪音通道 (CH1). 方波通道 (CH0) 接线不变, 见原版 wiring-table.md.

## 二、LFSR 结构 (HC374 八 D 触发器)

**与 HC164 的本质区别**: HC374 是边沿 D 触发器, D 输入独立; HC164 是串入并出, 只有 DSA/DSB 两个串入.
HC374 的每位 D 单独可控, 上电启动逻辑可强制 D[0]=1 (HC164 做不到, 它的串入由反馈独占).

**LFSR 接法 (XOR 反馈, max-length 8 位, 周期 255)**:

```
D[0] ← 启动逻辑 (RC 覆盖 + feedback, 见第六节)
D[1] ← Q[0]    (移位: 前级 Q → 下级 D)
D[2] ← Q[1]
D[3] ← Q[2]
D[4] ← Q[3]
D[5] ← Q[4]
D[6] ← Q[5]
D[7] ← Q[6]

feedback (CD4070 门1-3) = Q[7] ⊕ Q[5] ⊕ Q[4] ⊕ Q[3]
```

**工作过程**:
1. 上电: RC 脉冲维持 ~5ms, D[0] 强制=1, LFSR 在 noise_clk 驱动下被 1 填满
2. RC 充电后 (~5ms): D[0] 切回 feedback, max-length LFSR 自持 (非 0 永不归零)
3. 永不锁死

## 六、U12 — CD4070 (XOR 反馈)

> 用门 1-3 做 XOR 反馈, 门 4 闲置 (启动逻辑走 U4 HC00, 见第八节).
> CD4070 与 HC86 功能等效 (都是四 2 输入 XOR), 但**引脚布局不同** (CLAUDE.md 7.4 节).
> 本表据 CD4070 引脚, 不可混用 HC86 接线.

**CD4070 引脚** (据 TI CD4070B Rev.E DIP-14 datasheet, 85bb480 commit 核对):
```
Pin  1: 1A   Pin 14: VDD (+3~18V)
Pin  2: 1B   Pin 13: 4B
Pin  3: 1Y   Pin 12: 4A
Pin  4: 2Y   Pin 11: 4Y
Pin  5: 2A   Pin 10: 3Y
Pin  6: 2B   Pin  9: 3B
Pin  7: VSS  Pin  8: 3A
```

⚠️ 门 2/3 的 A/B/Y 脚位与 HC86 不同 (4000 系列典型布局, 门 2/4 的 A/B 顺序反).

**门分配**:
| 门 | 功能 | Pin (输入→输出) |
|----|------|----------------|
| 门1 | Q7⊕Q5 (xor_a) | A=1, B=2 → Y=3 |
| 门2 | Q4⊕Q3 (xor_b) | A=5, B=6 → Y=4 |
| 门3 | (Q7⊕Q5)⊕(Q4⊕Q3) = feedback | A=8, B=9 → Y=10 |
| 门4 | 闲置 | A=12, B=13 → Y=11 (接 GND, 悬空) |

**逐脚表**:

| Pin | 信号 | 连接 |
|-----|------|------|
| 1 (1A) | ← U13.Q7 (LFSR 第 7 位输出) | 门1 输入 a |
| 2 (1B) | ← U13.Q5 | 门1 输入 b |
| 3 (1Y) | xor_a (Q7⊕Q5) | → U12.8 (门3 输入 a) |
| 5 (2A) | ← U13.Q4 | 门2 输入 a |
| 6 (2B) | ← U13.Q3 | 门2 输入 b |
| 4 (2Y) | xor_b (Q4⊕Q3) | → U12.9 (门3 输入 b) |
| 8 (3A) | ← U12.3 (xor_a) | 门3 输入 a |
| 9 (3B) | ← U12.4 (xor_b) | 门3 输入 b |
| 10 (3Y) | **feedback (Q7⊕Q5⊕Q4⊕Q3)** | → **U4 门2 (HC00 反相输入, Pin 4,5)** |
| 12 (4A), 13 (4B) | → GND | 门4 闲置 |
| 11 (4Y) | — | 悬空 |
| 14 | VDD | +5V |
| 7 | VSS | GND |

## 七、U13 — 74HC374 (LFSR 主体, 替代 HC164)

> ⚠️ HC374 DIP-20, 比 HC164 (DIP-14) 多 6 脚, 面包板占位大.

8 位 D 触发器阵列, 共享 CP (noise_clk). /OE 接 GND (常输出).
所有 D 输入按 LFSR 移位规则连接 (D[i+1] ← Q[i]).

**74HC374 引脚** (Nexperia 74HC_HCT374 DIP-20):
```
Pin  1: /OE  Pin 20: VDD
Pin  2: Q0   Pin 19: Q7
Pin  3: Q1   Pin 18: Q6
Pin  4: Q2   Pin 17: Q5
Pin  5: Q3   Pin 16: Q4
Pin  6: D3   Pin 15: D4
Pin  7: D2   Pin 14: D5
Pin  8: D1   Pin 13: D6
Pin  9: D0   Pin 12: D7
Pin 10: GND  Pin 11: CP (上升沿触发)
```

| Pin | 信号 | 连接 | 说明 |
|-----|------|------|------|
| 1 (/OE) | → GND | 常输出 | |
| 11 (CP) | ← **U18.Y4 (HC00 绑定输出 noise_clk, Pin 11)** | LFSR 移位时钟 | |
| 9 (D0) | ← **U4.Y4 (HC00 门4 输出 = feedback OR RC启动, Pin 11)** | D[0] = 启动覆盖后的反馈 | ⭐ 核心 |
| 8 (D1) | ← U13.Q0 (Pin 2) | D[1] = Q[0] (移位) | |
| 7 (D2) | ← U13.Q1 (Pin 3) | D[2] = Q[1] | |
| 6 (D3) | ← U13.Q2 (Pin 4) | D[3] = Q[2] | |
| 15 (D4) | ← U13.Q3 (Pin 5) | D[4] = Q[3] | |
| 14 (D5) | ← U13.Q4 (Pin 16) | D[5] = Q[4] | |
| 13 (D6) | ← U13.Q5 (Pin 17) | D[6] = Q[5] | |
| 12 (D7) | ← U13.Q6 (Pin 18) | D[7] = Q[6] | |
| 2 (Q0) | → U13.D1 (Pin 8) | 反馈 (移位链) + 可接 U17 | |
| 3 (Q1) | → U13.D2 (Pin 7) | 移位链 | |
| 4 (Q2) | → U13.D3 (Pin 6) | 移位链 | |

| 16 (Q4) | → U13.D5 (Pin 14) + **U12.4 (HC86 门2 输入 a)** | 移位 + 反馈 tap | |

## 八、U4 借用 — HC00 门 2/3/4 (RC 启动逻辑)

> 方波通道 U4 (HC00) 门1 做 PE 反相 (v0.2 用), **门 2/3/4 借给噪音做 RC 启动**.
> 启动逻辑: `D[0] = feedback OR RC_pulse` (上电时 RC=1 强制 D[0]=1, 之后 RC=0 让 feedback 接管).
>
> OR 用 NAND 实现 (德摩根): `A OR B = NAND(NOT A, NOT B)` — 正好用 3 个 HC00 门.

**RC 电路** (上电脉冲, τ=4.7ms):

```
+5V ── C(0.1μF) ── 节点 ── R(47kΩ) ── GND

节点电压: 上电瞬间 = +5V (C 两端电压不能突变), 充电后 = 0V (R 拉低)
τ = R×C = 47kΩ × 0.1μF = 4.7ms
节点维持高电平约 5ms (足够 noise_clk 跑 20+ 拍填满 LFSR)
```

**门分配** (74HC00 DIP-14):

| 门 | 功能 | Pin (输入→输出) |
|----|------|----------------|
| 门1 | PE 反相 (v0.2 方波用, 不变) | 1,2 → 3 |
| 门2 | NOT(feedback) (输入短接当反相器) | 4,5 → 6 |
| 门3 | NOT(RC 节点) (输入短接当反相器) | 9,10 → 8 |
| 门4 | NAND(门2, 门3) = feedback OR RC | 12,13 → 11 |

**逐脚表**:

| Pin | 信号 | 连接 | 说明 |
|-----|------|------|------|
| 1 (A1), 2 (B1) | ← tc_hi (方波) | 门1 PE 反相 (v0.2 不变) | |
| 3 (Y1) | pe_n | → 方波通道 (v0.2 不变) | |
| 4 (A2), 5 (B2) | ← **U12.8 (HC86 feedback 输出)** | 门2 输入 (短接当反相器) | NOT(feedback) |
| 6 (Y2) | NOT(feedback) | → U4.13 (门4 输入 a) | |
| 9 (A3), 10 (B3) | ← **RC 节点** | 门3 输入 (短接当反相器) | NOT(RC) |
| 8 (Y3) | NOT(RC) | → U4.12 (门4 输入 b) | |
| 12 (A4) | ← U4.6 (NOT feedback) | 门4 输入 a | |
| 13 (B4) | ← U4.8 (NOT RC) | 门4 输入 b | |
| 11 (Y4) | **feedback OR RC (启动后反馈)** | → **U13.D0 (HC374 Pin 9)** | ⭐ 启动覆盖输出 |
| 14 | VCC | +5V | |
| 7 | GND | GND | |

**真值表验证**:

| RC 节点 | NOT(RC) (门3) | NOT(fb) (门2) | NAND (门4) | D[0] 含义 |
|---------|---------------|---------------|------------|-----------|
| 高 (上电, C 充电前) | 0 | x | NAND(x,0) = **1** | **强制灌种** ✅ |
| 低 (充电后, R 拉低) | 1 | NOT(fb) | NAND(NOT fb, 1) = **fb** | **正常反馈** ✅ |

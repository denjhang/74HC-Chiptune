# FSM Player - 双 ROM 自动播放器

## 架构说明

这是一个基于有限状态机(FSM)的双 ROM 自动播放系统，上电后自动从地址 0 开始播放 16-bit ROM 内容。

### 应用场景
- LED 点阵动画
- 简单音乐播放
- 波形输出
- 任意需要时序控制的场景

### 芯片清单
- 2 × 39SF040 (512KB Flash ROM)
  - ROM1: 程序 ROM (存储 FSM 指令)
  - ROM2: 数据 ROM (存储输出数据)
- 多个 74HC161 (4 位计数器，地址生成)
- 若干 74HC02/04/08/32 (逻辑门)
- 输出接口 (根据应用)

### 工作原理
1. 上电复位，FSM 进入初始状态
2. 从 ROM1 读取指令
3. 根据指令执行：输出数据、跳转、延时、循环等
4. 自动播放，无需外部输入

### 指令集设计 (待定)
- 输出指令：输出 16-bit 数据到端口

- 延时指令：等待 N 个周期
- 循环指令：条件跳转

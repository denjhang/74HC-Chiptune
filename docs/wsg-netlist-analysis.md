## 数据通路

```
CPU地址(AB3-0) ──┐
                  ├── U2(157) ──→ 3L-157-Y1~Y4 ──→ U4(189) 地址, U6(189) 地址
ROM地址(4H-32H) ─┘  sel = 2H (经 U3 86 XOR GND)

CPU数据(DB3-0) ──┐
                  ├── U5(158) ──→ 3K-158-Y1~Y4 ──→ U4(189) 数据, U6(189) 数据
174输出/ROM数据 ─┘  (反相输出, 抵消189反相输入)

U4(189) DO ──→ 2L-DQ0~DQ3 ──→ U8(283) A端
U6(189) DO ──→ 2K-DQ0~DQ3 ──→ U8(283) B端

U8(283) S1~S4,C4 ──→ U9(174) 锁存
U9(174) 1Q ──→ U8(283) C0 (进位反馈)
U9(174) 其他输出 ──→ U5(158) B端 (写回RAM)

U7(ROM) 数据 ──→ U9(174) / U5(158) B端
```

## 关键设计细节

### 157 (U4) 地址选择
- sel = `3LK-157-S = 2H XOR GND = 2H`
- 当 2H=0: 157 输出 CPU 地址 → 189 被 CPU 访问
- 当 2H=1: 157 输出 ROM 地址 → 189 被 synth 访问

### 158 (U5) 数据 MUX
- B端 = 174 输出 + 1M 波形ROM 输出 (线上短接)
- 反相输出 → 189 数据输入 (抵消 189 反相)

### 283 (U8) 加法器
- A端 = acc_do (2L-DQ), B端 = freq_do (2K-DQ)
- C0 = u9_q[0] (进位反馈)
- 输出 → 174 (U9) 锁存

### 174 (U9) 锁存
- D = {C4, S3, S2, S1, S0, 旧Q[1]} = 6-bit
- CLK = 3M-DQ0, nCLR = 3M-DQ3
- 输出: Q0→进位反馈, Q[5:2]→158 B端, Q5→1M 地址

### 189 (U6=acc, U7=freq)
- acc: nWE = cpu_wr1_n (CPU写)
- freq: nWE = ctrl_dq1 (3M ROM控制, 合成器回写)
- 输出反相, 连到 283 和 273

### 273 (U11) 输出锁存
- D = {acc_do[3:0], u10_dq[3:0]} = 8-bit
- 高4位 = acc (音量控制), 低4位 = 波形数据
- CLK = SOUND_ON

### 4066 (U12) 模拟音量
- 4个独立开关, 由 273 高4位控制
- 输入 = 波形数据, 通过电阻网络加权输出

### 4066 (U12) + 电阻网络
```

4066-O2 → R6.2, U12.2

```
- 当 2H=1（高地址区域）：157 输出 CPU 地址 → 189 被 CPU 访问
- 总线仲裁由 ROM 地址线本身决定，无需额外控制信号

### 158 反相 MUX
- 158 输出反相数据 → 189 数据输入
- 189 输出也是反相的（open-collector）
- 两级反相在数据通路中抵消，但中间信号是反相的
- 283 加法器接收的是反相数据？需要确认

### 189 输出反相
- 74LS189 输出是 open-collector 反相
- 写入 1011，读出 0100
- 需要在某处用异或门或反相器还原

### 174 六D触发器
- 锁存 283 加法结果（4-bit sum + carry）
- 一个输出反馈到 283 C0（进位链）
- 其他输出通过 158 回写到 189

### ROM 地址
- 32H,16H,8H,4H,2H,1H 来自 HCNT 计数器
- HCNT 是 6-bit 硬件计数器，6.144MHz
- 64 步/帧 = 96kHz 音频输出

## 27C256 引脚映射（网表实际连接）

基于网表中的 pin 编号反推：

**U7 (microcode ROM):**
- 地址: pin4=A7, pin5=A6, pin6=A5, pin7=A4, pin8=A3, pin9=A2, pin10=A1
- 数据: pin11=D0, pin12=D1, pin13=D2, pin15=D4(注意pin15)
- 控制: pin20=/CE, pin22=/OE
- 电源: pin1=VPP, pin14=GND(?), pin28=VCC

注意：U7 pin15 同时出现在 DQ3 数据线和 GND 网络中，封装编号需确认。

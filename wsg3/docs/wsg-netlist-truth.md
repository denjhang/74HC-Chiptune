# Pac-Man WSG 网表完整反推 (从 AD.net 一行行读)

## 网表信号清单

### 时序信号 (HCNT, 6-bit)
- 1H  = HCNT[0] — 从 U1.24 (排针) 进, 也接 U3.10 (ROM A1)
- 2H  = HCNT[1] — U1.23, U2.10 (异或门输入), U3.9 (ROM A2)
- 4H  = HCNT[2] — U1.22, U3.8 (A3), U4.13 (B4)
- 8H  = HCNT[3] — U1.21, U3.7 (A4), U4.10 (B3)
- 16H = HCNT[4] — U1.20, U3.6 (A5), U4.6 (B2)
- 32H = HCNT[5] — U1.19, U3.5 (A6), U4.3 (B1)

注: HCNT 由外部 (排针) 提供, 不是声卡内部产生!

### CPU 接口信号
- AB0-3 = U1.7,5,3,1 — CPU 地址低 4 位 → U4 A1-A4
- DB0-3 = U1.13,12,11,10 — CPU 数据低 4 位 → U5 A1-A4
- /WR0 = U1.15 → U3.4 (ROM /OE?)
- /WR1 = U1.16 → U6.3 (RAM1 /WE)
- SOUND_ON = U1.17 → U11.1 (HC273 /MR)
- /GM = U1.18 → U3.20 (ROM /CE)

## 芯片信号 → 引脚完整映射

### U2: 74HC86 (异或门) — DIP-14

网表连接:
- Pin 8 (Y3) → 3LK-157-S (U5.1, U4.1)
- Pin 10 (A4) ← 2H
- Pin 7 (GND), Pin 9 (B4) ← GND

**功能**: Y3 = A4 ^ B4 = 2H ^ 0 = 2H
所以 **3LK-157-S = 2H**

### U4: 74HC157 (4×2 选 1 mux, AB mux)
- Select = 3LK-157-S = 2H
- A1-A4 = AB0-3 (CPU 地址)
- B1-B4 = 32H, 16H, 8H, 4H (HCNT 高位)
- Y1-Y4 = 3L-157-Y1..4 → U6/U7 地址 A3-A0

注意! Y1 → A3, Y4 → A0 (顺序反)

当 Select=0 (2H=0): Y = A = AB (CPU 地址)
当 Select=1 (2H=1): Y = B = HCNT[5:2]

### U5: 74HC158 (4×2 选 1 反相 mux, DB 反相 mux)
- Select = 3LK-157-S = 2H
- A1-A4 = DB0-3 (CPU 数据)
- B1-B4 = 3K-158-B1..4 (反馈, 来自累加器/ROM)
- Y1-Y4 (反相) → U6/U7 数据输入

B 输入:
- B1 = 3K-158-B1 ← U10.6 (1M ROM D5), U9.14, U9.5
- B2 = 3K-158-B2 ← U10.7 (1M ROM D4), U9.7
- B3 = 3K-158-B3 ← U10.8, U9.10
- B4 = 3K-158-B4 ← U10.9, U9.12

注意 U9 是 174 锁存器, 输出 Q1-Q6

### U3: 27C256 / 82S126 (3M 微码 PROM)
地址 (8-bit):
- A0 = Pin 10 ← 1H
- A1 = Pin 9 ← 2H
- A2 = Pin 8 ← 4H
- A3 = Pin 7 ← 8H
- A4 = Pin 6 ← 16H
- A5 = Pin 5 ← 32H
- A6, A7 = 接 GND/VCC (不接 HCNT)
- /CE = Pin 20 ← /GM
- /OE = Pin 22 ← GND (常输出)

数据 (4-bit):
- DQ0 = Pin 11 → 3M-DQ0 → U9.9 (174 CLK)
- DQ1 = Pin 12 → 3M-DQ1 → U7.3 (U7 /WE)
- DQ2 = Pin 13 → 3M-DQ2 → U11.11 (273 CP)
- DQ3 = Pin 15 → 3M-DQ3 → U9.1 (174 /CLR)

注: DQ0=时钟, DQ3=清零, DQ1=RAM写, DQ2=输出锁存
原版 82S126 数据看 16 步循环 (HCNT[3:0])

### U6: 74LS189 (RAM #1, 存"2L"组数据)

地址 (4-bit, 共享):
- A0 = Pin 1 ← 3L-157-Y4 (U4 Y4)
- A1 = Pin 15 ← 3L-157-Y3
- A2 = Pin 14 ← 3L-157-Y2
- A3 = Pin 13 ← 3L-157-Y1

控制:
- /CS = Pin 2 ← GND (常使能)
- /WE = Pin 3 ← /WR1 (CPU 写脉冲)

数据输入 (4-bit, 共享):
- D3 = Pin 4 ← 3K-158-Y4
- D2 = Pin 6 ← 3K-158-Y3
- D1 = Pin 10 ← 3K-158-Y2
- D0 = Pin 12 ← 3K-158-Y1

数据输出 (4-bit, 反相, 独立):
- O0 = Pin 5 → 2L-DQ0 → U11.8, U8.6
- O1 = Pin 7 → 2L-DQ1 → U11.7, U8.2
- O2 = Pin 9 → 2L-DQ2 → U11.4, U8.15
- O3 = Pin 11 → 2L-DQ3 → U11.3, U8.11

**关键**: U6 输出 → U8 (283 加法器) A 端 + U11 (273) 高 4 位
所以 U6 存的是 **累加器/音量** 之类需要直接送加法器和输出锁存的数据

### U7: 74LS189 (RAM #2, 存"2K"组数据)

地址 (4-bit, 共享 U6):
- 同 U6

控制:
- /CS = Pin 2 ← GND
- /WE = Pin 3 ← 3M-DQ1 (微码控制写!)

数据输入 (4-bit, 共享 U6):
- 同 U6

数据输出 (4-bit, 反相):
- O0 = Pin 5 → 2K-DQ0 → U10.5 (1M ROM A4), U8.5 (283 B0)
- O1 = Pin 7 → 2K-DQ1 → U10.4, U8.3
- O2 = Pin 9 → 2K-DQ2 → U10.3, U8.14
- O3 = Pin 11 → 2K-DQ3 → U8.12 (注意! 只到 U8, 不到 U10)

**关键**: U7 输出 → U8 (283) B 端 + U10 (1M ROM) 地址低位
所以 U7 存的是 **频率/波形选择** (要送加法器和寻 ROM)

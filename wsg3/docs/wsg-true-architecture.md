# WSG 真实架构分析

## 技术文档关键信息

### 寄存器映射 (SPFM 总线)
- Voice 1: Waveform@5045, Freq@5050-5054 (20-bit), Vol@5055, Acc@5040-5044
- Voice 2: Waveform@504A, Freq@5056-5059 (16-bit), Vol@505A, Acc@5046-5049
- Voice 3: Waveform@504F, Freq@505B-505E (16-bit), Vol@505F, Acc@504B-504E

### 硬件工作原理
每 96kHz 时钟周期,对每个声道执行:
1. 20/16 位频率 + 累加器 (用 4 位加法器时分复用)
2. 用波形选择 (0-7) + 累加器高 5 位查 ROM
3. ROM 数据 × 音量 (0-15)
4. 输出到放大器

### 网表芯片清单 (11 片)
- U2: 74HC86 - 异或门 (地址译码)
- U3: 27C256 - ROM 1
- U4: 74HC157 - AB mux
- U5: 74HC158 - DB 反相 mux
- U6: 74LS189 - RAM 1 (2L 寄存器, 频率存储)
- U7: 74LS189 - RAM 2 (2K 数据, 累加器/音量)
- U8: 74HC283 - 加法器 (1K)
- U9: 74HC174 - 锁存器 (1L, 存进位)
- U10: 27C256 - ROM 2 (波形 ROM)
- U11: 74HC273 - 输出寄存器
- U12: CD4066 - 模拟混音

## 微码 ROM (3M) - U3: 27C256

**真实 ROM 芯片**,不是组合逻辑！
- 地址输入: 32H, 16H, 8H, 4H, 2H, 1H (来自 TDM 计数器)
- 数据输出: 3M-DQ0~3M-DQ3 (4 位控制码)
- 功能: 存储 16 步微码,控制 TDM 时序

## 波形 ROM - U10: 27C256

**真实 ROM 芯片**
- 地址输入: 波形选择[2:0] + 相位[4:0] = 8×32 = 256 字节
- 数据输出: 1M-DQ0~DQ3 (4 位波形数据)
- 功能: 存储 8 种波形,每种 32 采样点

## 加法器 (1K) 实现

```verilog
// 4 位加法器, 时分复用
wire [5:0] sum_1K = acc_wave_2K_dout + vol_freq_2L_dout + sum_d_1L[5];

// 1L 锁存进位
always @(posedge rom_3M_out[0] or negedge rom_3M_out[3]) begin
    if (rom_3M_out[3] == 0)
        sum_d_1L <= 0;
    else
        sum_d_1L <= {sum_1K, sum_d_1L[3]}; // 本次进位 + 4 位 + 上次高位
end
```

## 波形 ROM 地址

```verilog
// 1M: 波形 ROM 地址生成
wire [7:0] rom1m_addr;
assign rom1m_addr[7:5] = acc_wave_2K_dout[2:0]; // 波形选择 0-7
assign rom1m_addr[4:0] = sum_d_1L[4:0];         // 32 采样点
```
